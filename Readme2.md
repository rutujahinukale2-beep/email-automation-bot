# Email Automation Bot using Python

## Description
This project is a simple Python-based email automation bot that sends emails using the SMTP protocol.

## Features
- Sends email automatically using Python
- Uses SMTP (Gmail server)
- Error handling using try-except

## Technologies Used
- Python
- smtplib
- email.message

## How to Run
1. Clone the repository
2. Replace the sender email and App Password with your own credentials
3. Run the Python file

> Note: Gmail requires App Password and 2-Step Verification to be enabled.
